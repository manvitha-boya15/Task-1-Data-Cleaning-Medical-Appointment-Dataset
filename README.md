# Task-1-Data-Cleaning-Medical-Appointment-Dataset
Data Cleaning and Preprocessing of Medical Appointment No Shows Dataset using Excel

## Project Overview

This project focuses on cleaning and preprocessing the Medical Appointment No Shows dataset using Excel.

## Dataset

Medical Appointment No Shows Dataset

- Records: 110,527
- Columns: 14

## Data Cleaning Steps Performed

### Missing Values Check
- Checked all columns for missing values.
- No missing values found.

### Duplicate Records Check
- Checked for duplicate records.
- No duplicate records found.

### Column Name Standardization
- Renamed column headers using lowercase and underscores.

### Data Type Validation
- Verified numeric, text, and date columns.

### Date Format Review
- Reviewed and validated date columns.
